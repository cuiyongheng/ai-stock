## ⭐ 每月"投最便宜的"规则（核心动作）

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  每月发工资/有闲钱那天, 5 分钟:

  ① 拉 6 只现价 + 估股息率 + 52周分位 (Claude 用 ashare_data 帮拉)

  ② 排"便宜度"名次:
        便宜度 = 股息率越高(+) & 52周分位越低(+) & 基本面OK(必须)
     → 谁最便宜, 本月就投谁 (投 1-2 只即可, 别撒胡椒面)

  ③ 基本面闸门 (关键):
        买之前确认"便宜"是因为 板块回调/情绪, 不是 基本面变坏。
        基本面破了的便宜 = 陷阱, 不买。

  纪律:
    · 没投的钱放 货基/国债逆回购 (~1.5-2%) 当弹药, 越跌越买
    · 单只上限 ≤ 25-30%, 到顶就投第二便宜的
    · 分红再投入 (按同样"投最便宜"规则投回去)
    · 日常不看盘、不因股价波动调仓
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## ⭐ 仓位结构：底仓 + 弹药（2026-07-23 起 · 6 只通用）

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  每只 = 底仓(占坑滚分红) + 弹药(便宜才下重注):

  ┌─ 底仓 ~1/3 ───────────────────────────────────┐
  │  现在起, 小额、不问价、每月滚, 分红再投入        │
  │  目的: 占坑 + 复利先跑, 不踏空                  │
  └───────────────────────────────────────────────┘
  ┌─ 弹药 ~2/3 ───────────────────────────────────┐
  │  压住, 只在"便宜信号"触发才投, 越便宜越多投      │
  │  目的: 保留安全边际, 真便宜才重手                │
  └───────────────────────────────────────────────┘

  便宜信号 = 动态口径 (不再钉死绝对价, 避免"永远够不到"):
    加码档: 52周分位 < 40%   (每年通常有机会)
    重手档: 52周分位 < 20%   (逼近年内低点)
    闸门(必须): 分红未被砍 & 无红线 → 否则再便宜也不投

  每月动作: 底仓照投最便宜的1-2只; 触发加码/重手档 → 弹药跟上。
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 📋 巡检输出规范（会话开头 Step0 必带股息率）

固定列：`简称｜现价｜涨跌｜动态股息率｜52周分位｜加码价(<40%)｜重手价(<20%)｜信号`
- **排序**：按动态股息率从高到低 ＝ 便宜度排名，最高者＝本月最该投
- **信号列**：距加码档还差 X%（负＝还需下跌）；已进 <40% 分位标 ✅加码档、<20% 标 ✅✅重手档
- **加码/重手价**：巡检时用实时 52 周区间现算（低 ＋ 40%/20%×区间），不照抄快照
- **两种节奏**：新会话 Step0 = 2-3 行简报（排便宜度+是否触发）；每月定投 = 上面完整表

**股息率信号怎么用（关键：增量导向，不是卖出换仓）：**

```
✅ 股息率最高的1-2只 → 本月【新钱/分红】投向
✅ 某只息被爆炒到明显偏低 → 【新钱别投它】(不是卖它)
❌ 已持有的不因股息率高低卖出换仓
   —— 卖出触发红利税(短持10-20%)+择时, 破坏长持免税复利
   —— 存量只在【分红被砍(红线)/年度再平衡】时才动
```

## 全局触发规则

```
  提醒条件 (任一即在会话首条提醒):
    · 距某只财报披露日 ≤ 7 天
    · 现价接近"便宜度买点" (股息率升到吸引区)
    · 红线触发 (分红被砍风险 / 基本面恶化)

  只在这几种情况"动手", 其他都不动:
    1. 股息率极端偏高 + 基本面完好 → 加仓(用弹药)
    2. 年度再平衡 (权重漂移过大 → 修回)
    3. 红线触发 (减/换)
    4. 用新钱/分红 → 流向当前最便宜的
```
