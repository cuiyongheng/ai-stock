## Claude 使用本目录的方式

```
新会话开始时（讨论股票问题）:
  ↓
读取 watchlist/README.md
  ↓
扫一遍清单, 了解用户当前关注的股票范围
  ↓
讨论时优先围绕这些标的, 不在清单的股票要先确认是否要加入
```

## 单文件应该包含什么

1. 基本信息（代码、市场、行业）
2. 持仓情况（建仓价、数量、成本、浮盈亏）
3. 关注理由 (为什么持有)
4. 关键价位（加仓 Batch 1-4 / 减仓 / 止盈 / 止损）
5. 关键事件触发 (催化剂/风险)
6. 链接到 `reports/` 里的具体研究
7. 定期跟踪日期
