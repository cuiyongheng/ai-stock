## 文件命名约定

```
<6位代码>-<英文简称或拼音>.md
```

示例：
- `603986-gigadevice.md`
- `600519-moutai.md`
- `300750-catl.md`

## 单文件应该包含什么

1. 基本信息（代码、市场、行业）
2. 当前状态（持仓中/观察/已清仓）
3. 关注理由（1-3 句）
4. 关键价位（买入区间 / 卖出信号）
5. 链接到 `reports/` 里的具体研究/论文/财报跟踪
6. 最后更新日期

详细字段不强制——用户可以按自己习惯简化或扩展。
