## 一、核心论文（5 句话）

**我以 17.70 元 买入 上海医药 (601607)，因为：**

```
1. 这门生意的本质是 ── 医药商业(分销+零售)为主, 医药工业为辅,
   依靠国资体系的物流网络 + 招标关系赚取 1.65% 的稳定净利率

2. 它的护城河是 ── 行业 CR4 集中度 55%+, 全国分销网络与
   SPD 系统对医院形成一定锁定, 但护城河整体偏弱且持续变窄

3. 管理层 ── 上海实业集团(国资)控股, 现任董事长杨秋华 2024.3 上任,
   值得信赖的原因是 ── 国资背景规避了民营造假风险,
   但同时缺乏市场化激励, "稳"而不"卓越"

4. 当前价格相当于内在价值的 ── 约 1.0 倍 (合理区间中部),
   PE 11.5x / PB 0.87, 安全边际来自 ── ROE 与净资产支撑下行,
   PB 0.75 以下出现深度低估机会 (机械式估值锚, 非业绩驱动)

5. 即使我错了，下行风险可控，因为 ── 国资控股不会倒闭,
   PB 历史底约 0.55, 对应股价约 11 元 (向下空间 -38%);
   分红可对冲部分回报损失 (股息率 1.9-2.6%)
```

### 5 句话的问题暴露

```
✗ 第 1 句:  "本质是 1.65% 净利率" ← 这是接受平庸的论文,
              不是"伟大"论文
✗ 第 2 句:  "护城河偏弱且持续变窄" ← 这是反向论据,
              不是买入理由
✗ 第 3 句:  "稳而不卓越" ← 这是描述, 不是论文
✓ 第 4 句:  价格锚定还算合理
✓ 第 5 句:  下行可控勉强成立

整体判断: 这是一个"逻辑勉强自洽但根基薄弱"的论文
        在 17.70 元买入的真正理由可能是:
        - 行业龙头印象 (品牌)
        - 看着便宜 (PE/PB 低)
        - 国资央企 (稳)
        - 高股息预期 (实际仅 2.6%)
        - 听人推荐 / 跟风
        这些都不是足以支撑买入的核心逻辑
```

## 八、给持有人的诚实建议

```
你现在面临的不是"上海医药亏了 13.6%"的问题,
而是"我的本金被锁在一个年化期望回报只有 2.5% 的资产里"的问题

──────────────────────────────────────────────
机会成本对比 (假设本金 17.70 × 1000 股 = 17700 元)
──────────────────────────────────────────────
继续持有上海医药 (3 年期望年化 2.5%):
  3 年后 → 19,061 元   (净赚 1,361 元)

砍仓换恒瑞医药 (假设年化 12%):
  当前砍仓回笼 15,290 元
  3 年后 → 21,481 元   (净赚 4,191 元, 已扣浮亏)

差距: 2,830 元 (16% 仓位)
──────────────────────────────────────────────
```

**但是**——这只是数字模型，实际操作需要考虑：
- 你对恒瑞 / 迈瑞这类公司有没有研究信心？
- 砍仓的心理代价（确认损失 vs 浮亏可幻想回本）
- 仓位大小（小仓位砍掉无所谓，大仓位拉低收益）
- 你的整体组合多元程度

**最诚实的建议**：
1. **如果上海医药占你仓位 < 15%**：持有，专注研究下一只一流公司
2. **如果占比 15-30%**：考虑分批换股（每月减 1/3，3 个月换完）
3. **如果占比 > 30%**：必须减仓，集中度过高
