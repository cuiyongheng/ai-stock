## 一、核心论文（5 句话）

**我以 36.00 元 (分批) 买入 招商银行 (600036)，因为：**

```
1. 这门生意的本质是 ── "卖信任" 的零售银行,
   通过 IT 系统 + 客户经理 + 私行体系, 把零售客户与
   高净值客户深度锁定, 赚取利差 + 财富管理费的双重收入

2. 它的护城河是 ── 27 年品牌沉淀 + 19.93 万私行客户 +
   全行业最强 IT 系统, 三重叠加在国内难复制,
   而且在持续变宽 (财富管理 AUM 17 万亿, +18% YoY)

3. 管理层 ── 招商局集团 (央企) 控股 + 市场化管理层,
   王良 (老招行 27 年) 平稳过渡到王小青 (内部培养),
   值得信赖的原因是 ── 体系化运作, 不依赖单一个人,
   田惠宇事件已处置且未影响业务延续

4. 当前价格相当于内在价值的 ── 0.7-0.8 折 (合理偏低),
   PE 6.02 / PB 0.80 位于历史 30 分位,
   安全边际来自 ── 拨备覆盖率 411% (行业前三) +
   股息率 5.56% (即使股价 0 增长, 5 年累计回报 28%)

5. 即使我错了，下行风险可控，因为 ── 央企背景不会倒闭,
   历史 PB 底约 0.55 (对应股价约 25 元, 向下空间 -30%);
   高拨备已为房地产风险充分计提;
   分红可对冲大部分下行损失
```

```

✓ 第 2 句: 三重护城河量化, 不是空话
```
