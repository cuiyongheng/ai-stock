## 2. 生意本质分析（段永平视角）

### 2.1 一句话定义

> **"长江电力靠中国最稀缺的水电资源（三峡为首的 6 座梯级电站）卖电赚钱，本质是'100 年期的现金流债券+轻微通胀对冲'。"**

### 2.2 收入结构（2024）

```
                              营收 845 亿 (2024)
                            ┌─────┴─────┐
                            │           │
                  电力业务 745 亿     其他 100 亿
                       (93%)           (12%)
                  毛利率 62.51%      投资收益+辅助业务
                          │
                          │
                  ┌───────┴──────────┐
                  │                  │
              六大电站            售电
              (装机 71.7 GW)      (上网电价 285.52 元/MWh)
                  │                  │
            ┌─────┴─────┐         电网长协 80%
            │           │         市场化交易 20%
        长江干流        金沙江
      ─────────────  ─────────────
      三峡 22.5 GW    乌东德 10.2 GW
      葛洲坝 2.7 GW   白鹤滩 16.0 GW
                     溪洛渡 13.9 GW
                     向家坝 6.4 GW
```

**关键事实**：
- 装机 71.795 GW = 中国总水电装机的 16.45%
- 是世界最大水电运营商
- 这些电站的水权 + 运营权 = **不可复制的稀缺资源**

### 2.3 商业模式画布

| 维度 | 评价 | 说明 |
|------|------|------|
| 一次性 vs 复购 | **永续复购** ✓ | 电网必须用，无法替代 |
| 锁定强度 | **极强** | 长协合同 + 输电通道独占 |
| 经营杠杆 | 极高 | 变动成本接近 0（水免费） |
| 资本密集度 | 极高 (前期) | 但电站建成后边际成本极低 |
| 议价能力 | 中 | 上网电价由国家核定，不能随意涨 |
| 现金流稳定性 | **极高** ★★★★★ | 经营现金流/净利 > 2.0 |

### 2.4 经营杠杆分析

```
═══════════════════════════════════════════════════
  长电的"成本结构" 特殊性
  ─────────────────────────────────────────────
  
  变动成本 ≈ 0   (水流过坝, 不要钱)
  固定成本:
    ├── 折旧摊销 ~250 亿 (前期建设投入)
    ├── 财务费用 ~111 亿 (债务 2995 亿, 平均利率 3.2%)
    └── 人工运营 ~30-50 亿
  
  → 每度电卖 0.285 元
    净利率 = 净利 345 / 营收 862 ≈ 40% (行业最高)
  → 1 千瓦时 = 0.115 元的纯利润
  → 一年 2959 亿千瓦时 → 一年 340 亿净利
  
  关键: 折旧到期后, 同样的发电量 = 更多净利
       → 这是"未来 10 年的内生增长"
═══════════════════════════════════════════════════
```

### 3.2 100 亿能否复制？

```
═══════════════════════════════════════════════════════
  ❌ 绝对不能

    1. 水电资源已开发完毕 (中国主要大江主流已建)

    3. 新建大型水电的环评 + 移民 + 生态成本极高
    4. 水权审批是政府特许, 不卖
    5. 输电通道独占, 国家电网长协绑定
    
  → 100 亿连一个白鹤滩 (1500 亿) 的 7% 都买不到
```

### 3.4 巴菲特式追问

> **"10 年后这条护城河还在吗？"**

**几乎肯定还在**。摧毁因素几乎不存在：

```
✅ 极有可能仍在:
  1. 长江还会流, 电还要用
  2. 中国主要水电资源不会突然多出来
  3. 三峡集团央企控股, 不会倒闭
  4. 折旧到期后, 利润反而会释放更多

⚠️ 可能变窄的小概率事件:
  1. 极端气候变化 → 长江流量减少
     (但短期 10 年内基本不可能)
  2. 核聚变商业化 → 电力价格归零
     (但 10 年内做不到)
  3. 抽水蓄能技术革命 → 不再需要水电
     (但抽蓄本身也是长电业务)
```

**这是巴菲特最爱的"护城河变宽 + 不会消失"的标的之一。**
