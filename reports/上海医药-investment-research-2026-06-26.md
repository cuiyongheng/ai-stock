## 2. 生意本质分析（段永平视角）

### 2.1 一句话定义

> **"上海医药是一家做药品物流+零售的国资央企，副业做仿制药与少量创新药。本质是低毛利通道生意，不是药企。"**

### 2.2 收入结构（2024）

```
                              营收 2752 亿
                            ┌─────┴─────┐
                            │           │
                    医药商业 2515亿      医药工业 237亿
                       (91%)              (9%)
                    毛利率 6.4%         毛利率 59.2%
                          │                  │
              ┌─────┴─────┐           研发投入
              │           │              28.18 亿
           分销           零售           (营收 1%)
          约 2300亿      约 215亿
         毛利率 5.94%    毛利率 12.84%
```

**结构性问题**：
- 91% 的收入来自低毛利分销，**整体毛利率被锁死在 10-12%**
- 工业（药品制造）是真正的利润板块（贡献近一半利润），但规模太小
- 即使工业板块翻倍到 470 亿（2030 年目标），分销占比仍 > 80%

### 2.3 商业模式画布

| 维度 | 评价 | 说明 |
|------|------|------|
| 一次性 vs 复购 | **高复购** ✓ | 医院/药店刚需采购 |
| 锁定强度 | 中 | 物流网络+SPD 系统切换成本不高，但医院招标多年合同 |
| 经营杠杆 | 极低 | 营收每增 100 元，净利仅增 1.65 元 |
| 资本密集度 | 中 | 仓储+车队+应收账款占用大量营运资本 |
| 议价能力 | 弱 | 上游药企（带量采购后量价齐压）+ 下游医院（结款慢） |

### 2.4 段永平式追问

> **"这门生意好在哪？"**

不"好"，只是"稳"。优点是：
- 行业必需（医保支付的钱必须通过分销网络流到药店和医院）
- 集中度高（CR4 > 55%），玩家有限
- 现金流稳定（OCF/NI 长期 > 1.0）

缺点是：
- 没有定价权
- 没有规模回报（规模越大 ROE 越低，"规模与回报倒挂"）
- 永远是"赚 1-2% 净利率"的生意

**结论：能存活，不能"伟大"**。段永平大概率会说：「这不是我要的生意。」

## 3. 护城河评估（巴菲特视角）

### 3.1 五类护城河逐项

| 类型 | 验证 | 强度 |
|------|------|------|
| 品牌/定价权 | 工业板块部分子公司（如信谊、华氏大药房）有品牌，但分销板块完全没有 | **弱** |
| 转换成本 | SPD 系统对医院有一定锁定（IT 接入），但商业谈判仍可换 | **中弱** |
| 网络效应 | 物流网络越大配送越便宜，但有规模天花板 | **中** |
| 规模效应 | 是行业第二大，2300 亿分销规模，但**ROE 反而低于民营九州通** | **不成立** |
| 技术/专利 | 工业板块少量创新药管线（如 SPH3127、SPH5030），规模微不足道 | **弱** |

### 3.2 护城河趋势

```
2018 ─────────── 2020 ─────────── 2024 ─────────── 2028(预测)
  │               │                  │                │
带量采购前        集采开始            集采常态化      仿制药出清
分销毛利率 8%+    毛利率 7%           毛利率 5.94%    继续承压
                                     ↓
                              护城河持续变窄
```

**核心问题**：分销业务在带量采购后毛利被压缩了 25%+，未来无反转空间。

### 3.3 巴菲特式追问

> **"10 年后这条护城河还在吗？什么能摧毁它？"**

护城河会在但会**更窄**。摧毁因素：
1. 医保 DRG/DIP 改革进一步压价
2. 药企直供医院（绕过分销）的政策放开
3. 电商医药 B2B 平台（如京东健康、九州通"好药师"）侵蚀基层市场

护城河变宽的可能：政策强制提高分销集中度（CR4 → CR10）。但**这是政策博弈，不是商业能力**。

## 4. 逆向思考与风险清单（芒格视角）

### 4.1 失败路径

| 路径 | 概率 | 影响 | 说明 |
|------|------|------|------|
| 集采进一步压缩毛利 | **高** | 净利 -15% | 工业板块仿制药仍在集采池 |
| 应收账款减值 | 中 | 净利 -10% | 截至 2024Q3，行业应收 2937 亿，上药占比约 25% |
| 国资改革停滞 | 中 | 长期估值锁死 | 国资控股，缺乏管理层激励 |
| 工业板块创新药失败 | 中 | 长期增长无 | SPH3127 等核心管线临床进度受质疑 |
| 利率上升 | 低 | 财务费用 +5 亿 | 短期借款体量大（>300 亿） |
| 大股东减持 | 极低 | 股价压力 | 上实集团目前在 H 股增持 |

### 4.2 历史类比

```
全球医药分销巨头 ROE 对比 (近 10 年均值)
═══════════════════════════════════════
  McKesson (美国)        14-18%   ← 美国行业 CR3 = 90%+, 寡头垄断
  Cardinal Health (美国) 10-15%
  AmerisourceBergen      12-16%
  Walgreens Boots (英美)  8-12%
  ──────────────────────────
  国药控股 (中国)         7-9%
  上海医药 (中国)         6-10%   ← 当前 7.74%

```

**结论**：中国医药分销 ROE 系统性低于美国，因为：
- 中国分销 CR4 = 55%（美国 90%+），竞争更激烈
- 应收账款回款周期 100+ 天（美国 35 天）
- 仿制药占比高（毛利天花板低）

**这是行业结构问题，不是公司经营问题**。

### 4.3 芒格式追问

> **"聪明人为什么会不买这家公司？"**

PE 9.9x、PB 0.75x 看着很便宜，但市场聪明的资金不买，因为：

1. **没有增长**：5 年净利复合增速仅 2.3%
2. **没有 alpha**：行业逻辑已被充分定价
3. **机会成本高**：同样 PB 0.75 可以买招商银行（ROE 14%+）、长江电力（ROE 16%+）
4. **国资属性**：分红率 23%，远低于市场对国资央企的 40-50% 期望
5. **看空者已经空过了**：H 股长期折价 35%+，市场用脚投了票

### 7.2 反向 DCF（隐含增长）

```
当前股价 15.29 / EPS 1.54 = PE 9.93x

若 PE 长期回归 10x (合理值):
  那么市场隐含 EPS 长期年增 = 0% (零增长)

若 PE 回归 12x (略乐观):
  那么市场隐含 EPS 长期年增 = -2% (轻微下滑)

→ 市场没有给"任何增长溢价"
→ 估值合理但不便宜, 因为生意本身没有增长
```

### 7.3 三情景估值（financial_rigor.py 验算）

```
三情景估值 (3年期, EPS 1.54 起算)
═══════════════════════════════════════════
情景        年增速   目标PE   目标EPS  目标价   涨跌
─────────────────────────────────────
乐观  Bull   10%    14x     2.05    28.7   +88%
中性  Base    4%    10x     1.73    17.3   +13%
悲观  Bear   -2%     7x     1.45    10.1   -34%
```

**风险收益不对称**：

```
                ↑ 收益
                │
       +88% ────●  (乐观, 概率 15%)
                │
                │
       +13% ────●  (中性, 概率 50%)
                │
       ────────┼─────→ 概率
       -34% ────●  (悲观, 概率 35%)
                │
                ↓ 损失

期望回报 = 0.15×88 + 0.50×13 + 0.35×(-34)
        = 13.2 + 6.5 - 11.9
        = +7.8% (3年累计, 年化 2.5%)
```

**期望年化回报 2.5%，低于无风险利率（10 年国债 2.0-2.5%），毫无吸引力**。

### 7.4 段永平式追问

> **"如果股市明天关闭 5 年，你愿意以这个价格持有吗？"**

**不愿意**。理由：
- 5 年累计回报很可能不超过 30%（年化 5-6%）
- 持有期间无法享受复利（ROE 仅 7.7%，且分红率低）
- 5 年后可能仍要担心集采、DRG、应收账款问题
