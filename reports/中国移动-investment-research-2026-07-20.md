## 五、用户 5 大关注点 · 逐条回答

### ① 分红可持续性（派息率承诺）—— 强

```
  · 派息率: 公司承诺逐年提升, 目标 3 年内升至 ≥75% (2023年宣布)
    2024 派息率约 73%, 呈上升趋势
  · 关键含义: 即使利润零增长, 派息率上升 → 每股分红仍能【小幅增长】
    → 这是"分红成长"的来源, 比格力那种"高但停滞"的分红更健康
  · 置信度: 中高 (承诺明确, 但具体派息率需以年报为准)
```

### ② 现金流质量（自由现金流覆盖分红）—— 强

```
  · 经营现金流极强 (估~2800-3000亿/年), 资本开支~1700亿/年
    → 自由现金流估~1000亿+, 覆盖年分红(~1000亿) 约 1 倍
  · 现金储备巨大 (估3000亿+现金及存款, 净现金)
  · 分红发得出、发得稳 —— 这是它比中字头(现金差)强的核心
  · 置信度: 中 (ashare_data 无现金流表, 数据为行业经验估算, 需年报核实)
```

### ③ A股 vs H股(00941)溢价 —— A股明显贵

```
  A股 600941: 95.80, PE 15.3, 估股息率 ~4.8%
  H股 00941 : 约 HK$85-90, PE ~11, 估股息率 ~5.8-6.5% (估算)
  → A股比H股贵约 30%+, 股息率低约 1 个百分点
  → 若你有港股通(门槛50万): 买 H股 更划算(便宜+高息)
    代价: H股红利税规则不同 + 汇率波动
  · 置信度: 中 (H股价格为估算, 需实时核实)
```

### ④ 算力/移动云第二曲线 —— 有, 但撑不起高成长

```
  · 移动云收入 估~1000亿+ (2024), 增速20%+, 是增长最快板块
  · 算力网络/AIDC 投入加大, 政企数字化(DICT)是主引擎
  · 但: 占总收入仅~25%, 且云业务毛利低于主业
    → 能"对冲"手机饱和, 撑不起"整体高成长"
  · 定位: 从"故事"看是二曲线, 从"财务"看还不足以扭转增长停滞
```

### ⑤ 增长天花板 + 资本开支 —— 见顶, 但capex拐点是利好

```
  · 增长天花板: 手机/宽带饱和(用户见顶), 收入增速已降到~1%
  · 资本开支: 5G 建设高峰已过, capex 见顶回落趋势
    → capex 下降 = 自由现金流改善 = 分红空间更大 (这是隐藏利好)
  · 净判断: 增长没了, 但"少花钱多分红"的公用事业阶段对收息者友好
```
